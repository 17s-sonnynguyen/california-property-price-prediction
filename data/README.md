# Data Directory

This directory stores datasets used throughout the project.

## Important Notice

The raw datasets are not included in this repository.

Data is obtained through a private FTP server provided for internship purposes and may contain proprietary information. As a result, raw data files are excluded from version control and should not be uploaded to public repositories.

## Expected Files

Examples:

* CRMLSSold202512.csv
* CRMLSSold202601.csv
* CRMLSSold202602.csv
* CRMLSSold202603.csv
* CRMLSSold202604.csv
* CRMLSSold202605.csv

## Metadata

Additional field definitions and dataset documentation are provided separately through:

* Trestle Property MetaData.pdf

## Usage

Place downloaded CSV files inside the local data directory before running notebooks or scripts.

Example:

```text
data/
├── CRMLSSold202512.csv
├── CRMLSSold202601.csv
├── CRMLSSold202602.csv
├── CRMLSSold202603.csv
├── CRMLSSold202604.csv
└── CRMLSSold202605.csv
```
